# ADR-008: Project Licensing — Proprietary

Status: Accepted
Date: 2026-02-19
Deciders: @owner

## Context

Необходимо определить лицензию проекта. Data-First стратегия предполагает монетизацию через данные и premium-функции. BirdNET используется под CC BY-NC-SA 4.0.

## Decision

**Proprietary (closed source).**

- Код приложения — закрытый, проприетарный
- Репозиторий — приватный (GitHub private)
- BirdNET модель — используется под CC BY-NC-SA 4.0 (некоммерческое) на этапе разработки и тестирования
- Перед коммерческим релизом необходимо решить вопрос лицензии ML-модели (см. ADR-003)

## Consequences

### Positive
- Полный контроль над коммерческим использованием кода
- Защита Data-First стратегии — конкуренты не могут форкнуть приложение
- Свобода в выборе модели монетизации

### Negative
- Нет вклада от open-source сообщества
- Нет бесплатного CI через GitHub Actions для public repos (2000 мин/мес для private)

### Neutral
- Собранные пользователями данные регулируются отдельно через Terms of Service / EULA
- Лицензия может быть пересмотрена позже (перейти на AGPL, dual licensing и т.д.)

## Alternatives Considered

- **GPL v3** — защита от коммерческого копирования, но код открыт. Отклонено: конкуренты могут взять архитектуру и создать аналог
- **MIT/Apache 2.0** — максимальная открытость. Отклонено: противоречит Data-First стратегии и монетизации
- **Отложить решение** — работать без лицензии. Отклонено: лучше определиться сейчас для ясности
